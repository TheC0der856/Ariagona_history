Create a PCA: To run the code a vcf file is needed called all.islands.DP10.maxmiss0.8.thin100.recode.vcf, containing information about all islands. A table is needed containing the information of Ariagona collection points called Ariagona_margaritae_Alles.csv. Furthermore popmap_all_islands_t_h_g.txt is used to devide the datasets into the islands. This file contains two columns. The first column contains Specimen IDs. The second column contains the letters "t" for Tenerife, "h" for El Hierro, and "g" for La Gomera. Both columns are tab separated. Another file called ID_Pop_noOutgroup.txt is used to devide the specimens into groups within the islands. It also contains two columns, one for specimens and one for place names, and is tab separated. The placenames are the same as used for Stacks in Gronefeld et al. 202X. All these files should be saved in a folder called data.


Create Stairwayplots: After running StairwayPlot download the folder on your local computer and insert the stairway_plot.R file in this folder. Now the code can be applied on the data.
